How This Works Plain-English guidance for the settings most operators use when shaping the bot’s behavior.
Investment Style
Aggressive seeks more participation and accepts higher volatility. Neutral is the balanced default. Conservative deploys more slowly and keeps more cash protection. Safe Mode / All Cash stages a defensive no-new-risk posture, but it does not automatically liquidate current holdings.
Max Buy Per Trade
Caps how much quote currency the bot can commit in a single order attempt. Higher values let the bot move faster, while lower values slow position building and reduce per-trade exposure.
Minimum Cash Reserve
Keeps a portion of the portfolio in cash after trades. Raising it makes the bot more defensive; lowering it allows more capital to deploy into ideas.
Core / On / Auto / Off
Core assets are strategic holdings with target weights and rebalance bands. On manually allows a non-core asset. Auto lets the system decide. Off blocks an asset from manual participation while keeping it visible.
Sniper Score and Buy Scale
Sniper minimum score controls how selective the opportunistic entry engine is. Sniper buy scale controls how much capital a qualifying sniper entry may use relative to normal sizing.
Drawdown Levels
Warn, Reduce, and Freeze tell the bot how defensive to become as the portfolio drawdown deepens. Lower thresholds react earlier; higher thresholds tolerate more drawdown before tightening behavior.
